## 重要说明

硬件要求：

    仅支持一块 500G 以上的 SATA 硬盘
    需要一个 4G 以上的 U盘
    需要 CentOS 将硬盘识别为 sda，将U盘识别为 sdb

## 制作 Myudisk

1. 制作启动U盘：

使用 [老毛桃U盘启动盘制作工具][3] 将U盘制作成启动U盘，并且将其卷标命名为 `QUENONG`。
